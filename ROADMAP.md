# AlphaCut Roadmap

This file tracks active and proposed work. Completed delivery history lives in
`COMPLETED.md`, and research notes live in `RESEARCH_REPORT.md`.

## Active Queue

- [ ] Inno Setup installer for Windows
- [ ] ROI selection and manual mask painting
- [ ] Multi-GPU support
- [ ] Drag-out support for moving the output file into an NLE
- [ ] Localization scaffold for i18n-ready strings
- [ ] Thumbnail grid for batch preview
- [ ] Virtual camera output via `pyvirtualcam` for OBS and video calls
- [ ] FFmpeg-pipe RGB24 stdin mode for scriptable pipelines
- [ ] MatAnyone foreground/alpha pair export for NLEs without alpha support

## Candidate Enhancements

- [ ] RVM temporal-coherent model path for live/interview footage
- [ ] Interactive refine-mask brush with one-frame corrections propagated across adjacent frames
- [ ] Optical-flow mask propagation when frame skipping
- [ ] ONNX plus NCNN Vulkan dual backend for Apple Silicon and AMD users
- [ ] Model-registry JSON declaring model name, URL, hash, size, and license
